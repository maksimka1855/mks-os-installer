# MKS OS INSTALLER

Console installer for MKS OS distributions.

## Quick Start
Download `MKS OS INSTALLER.exe` and run it.

## Requirements
- Windows 10/11
- .NET 10 Runtime

## For Developers
cd Src
dotnet build
## Build from Source
dotnet publish -c Release -r win-x64 --self-contained true
## Download Links
The application provides direct access to:
